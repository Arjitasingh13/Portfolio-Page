# 🌟 Arjita Singh Portfolio

A modern, responsive, and interactive personal portfolio website showcasing my education, skills, projects, certifications, and contact information.

## 🚀 About the Project

This portfolio website was built to present my journey as a Front-End Developer and Computer Science student. It highlights my academic background, technical skills, personal projects, certifications, and provides visitors with an easy way to connect with me.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern dark-themed user interface
* Smooth navigation with fixed header
* About Me section with profile image
* Education timeline
* Skills showcase
* Project gallery with descriptions
* Certifications section
* Contact form with validation
* Email submission support using FormSubmit
* Share and Copy Portfolio Link functionality
* Browser notification support for contact submissions
